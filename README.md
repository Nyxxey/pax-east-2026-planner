# PAX East 2026 Planner

An interactive web app for navigating PAX East 2026 — featuring a browsable map and a sortable spreadsheet view to keep track of booths, panels, and points of interest.

## Features

- 🗺️ Interactive map view of the show floor
- 📊 Spreadsheet view for organizing booths, events, and schedule
- 🔀 Tabbed interface to switch between views
- 🎨 Custom dark-themed UI

## Built With

- HTML
- CSS
- JavaScript

## Usage

Open `pax-east-2026.html` in any modern web browser — no build step or dependencies required.
